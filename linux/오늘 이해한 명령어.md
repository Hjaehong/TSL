# RPM 
> 패키지 관리 명령어
- -U : (upgrade) 설치된 패키지에서 새로운 버전으로 업그레이드
- -v, --verbose : rpm 버전정보 한줄 출력
- -h, --hash : 묶인 패키지를 풀고 50개 해쉬 마크를 출력
- --force : **replacepkgs** 와 **replacefiles** 같이 동일하게 동작하는데 이미 일부 파일이 설치되어도 덮어서 파일을 설치한다.

# cp
> 복사
- -r : 밑의 디렉토리까지 전부 복사
- -f : 열수 없거나 제거 되었을 때 무시
- -a : 묶어서 복사(archive)
